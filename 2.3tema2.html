HTTP con Postman

1- Crear y ejecutar la siguiente petición HTTP:
GET /api/?gender=female HTTP/1.1
Host: randomuser.me


2- Crear mensaje HTTP tipo GET que ejecute la siguiente instrucción:
curl https://rickandmortyapi.com/api/character/


3- Crear y ejecutar el siguiente mensaje tipo POST:
curl --location --request POST 'http://httpbin.org/post' \
--form 'name="Joan"' \
--form 'email="joan@gmail.com"'

app.get('/saludo/', (req, res) => {
const cadena = `Hola! ${req.query.nombre}`
res.send(cadena)
})

console.log('Ir a dirección: localhost:5000/saludo/?nombre=tunombre'))
