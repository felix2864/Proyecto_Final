Ejercicios Memorias

1- Calcula el ancho de banda de una memoria DDR3 - 1600

Velocidad de la memoreia 1600 MHz -- dos transferencias de datos por ciclo de reloj -- transfiere 64 bits de datos simultáneamente.
Ancho de banda = Velocidad de datos × Transferencias por ciclo × Ancho de interfaz
Ancho de banda = (1600 MHz × 2 × 64 bits) / 8 bits/byte / 1024 bytes/kilobyte / 1024 kilobytes/megabyte =25.6 GB/s
El ancho de banda serian de 25.6 GB/s


2- ¿Qué jerarquía de memoria sigue un PC? Explica brevemente en cada niverl el tipo de memoria que lo forma.

Memoria caché:La memoria caché almacena datos y instrucciones utilizados con frecuencia para mejorar el rendimiento del sistema.
Memoria principal(RAM):Es donde se almacenan los datos y programas en uso activo por el sistema operativo y las aplicaciones en ejecución.
Almacenamiento segundario(disco duro):Estos dispositivos ofrecen mayor capacidad de almacenamiento en comparación con la memoria principal, pero son más lentos en términos de velocidad de acceso y transferencia de datos.


3- ¿Qué es el ancho de banda?¿y la latencia?

Ancho de banda:Es la capacidad máxima de transferencia de datos de una red o de un canal de comunicación en un período de tiempo determinado.
Latencia:Es el tiempo que tarda un paquete de datos en viajar desde su origen hasta su destino a través de una red.


4- ¿Qué tipos de memorias externas existen?¿Qué caraterísticas definen una memoria EPROM?

Tipos de memoria externas:
Memoria flash:Es una memorai no volatil que se utiliza en dispositivos de almacemaniento como tarjetas de memoria, USB y SSDs.
Disco duro:Es una memoria magnetica se utiliza para almacenar datos a largo plazo en PC o en otos dispositivos.
Memoria óptica: Incluye formatos como CDs, DVDs y discos Blu-ray. Estas memorias utilizan tecnología láser para leer y escribir datos en discos ópticos que contienen capas reflectantes y no reflectantes.
Memoria magneticas:Incluyen cintas magnéticas y tarjetas de banda magnética. Estas memorias utilizan campos magnéticos para almacenar y leer datos.
Memoria EPROM:
(Erasable Programmable Read-Only Memory), se trata de un tipo de memoria no volátil que puede ser programada y borrada eléctricamente.


5- ¿Qué información se proporciona cuando se dice que he adquirido una memoria DDR400?

-Tipo de memoria -Frecuencia -Ancho de banda -Compatiblidad
Estas especificaciones son importantes para asegurarse de que la memoria sea compatible con el sistema y para determinar su rendimiento potencial.


6- ¿Qué mejoras presenta la memoria DDR3?

-Mayor velocidad de transferencia -Mayor capacidad y densidad -Menor consumo de energía -Mejoras en la integridad de la señal
La memoria DDR3 ofrece una combinación de mayor velocidad, capacidad, eficiencia energética y confiabilidad en comparación con las generaciones anteriores de memoria.


7- ¿Qué es un módulo de memoria?¿Que tipo es el más adecuado actualmente?

Un módulo de memoria es una una placa o tarjeta que contiene chips de memoria integrados que se utiliza para almacenar datos en un PC.
El tipo de memoria mas adecuado dependeria del PC, los sistemas más nuevos y de alta gama suelen ser compatibles con las últimas generaciones de memoria, como DDR4. El sistema es más antiguo, puede ser compatible con DDR3 o DDR2.


8- ¿Qué características tiene el encapsulado de chip tipo FBGA?

-Diseño compacto -Alta densidad de pines -Distribución uniforme de pines -Conexiones de bola -Mayor rendimiento térmico
El encapsulado FBGA ofrece ventajas en términos de densidad de componentes, rendimiento térmico y diseño compacto, lo que lo hace adecuado para aplicaciones que requieren alta capacidad de almacenamiento y procesamiento, como las memorias RAM y las unidades de estado sólido (SSD).
