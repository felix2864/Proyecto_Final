Tema 3: CPU

Cómo funciona una CPU

Ejercicios CPU

1- ¿Qué diferencia existe entre CPU y microprocesador?¿Que partes forman la CPU?

La CPU: tiene una unidad de control , una unidad de registros, lógica y aritmética, y una memoria caché.
El microprocesador: son pequeños dispositivos electrónicos con carga eléctrica, tienen un interruptor de encendido/apagado
Partes que forman una CPU: Núcleo, unidad de control, unidad aritmética lógica, unidad de coma flotante, memoria caché, registros, controlador de memoria, bus.


2- ¿Qué función tiene el registro Acumulador de la ALU?¿Y el reloj de la UC?

Registro Acumulador de la ALU: Tiene como función hacer un registro almacenan temporalmente resultados aritméticos y lógicos intermedios.
Reloj de la UC: Emite una serie de pulsos electrónicos e intervalos constantes llamados ciclos que marcan el ritmo para la realización

4- ¿Que diferencia existe entre una unidad de control microprogramada y una unidad de control cableada?Justifica tu planteamiento ¿A que nos referimos cuando decimos que el juego de instrucciones de RISC?

Unidad de control cableada: Es difícil de modificar porque las señales de control que deben generarse están cableadas.
Unidad de control microprogramada: Es fácil de modificar por que la modificación se debe hacer solo en el nivel de instrucción


5- ¿Qué es el direccionamiento?¿Que tipos existen?Explica brevemente cada uno de ellos.

Es la configuración de la red en que a cada equipo le corresponde una única dirección para que no haya errores para transmitir datos.
Tipos Direccionamiento de registro: Las instrucciones acceden a los registros internos de la CPU para realizar operaciones y almacenar resultados.
Direccionamiento inmediato:Los operandos necesarios para ejecutar una instrucción se encuentran directamente en la instrucción misma.
Direccionamiento directo:Las instrucciones acceden directamente a una dirección de memoria específica para obtener o almacenar datos.
Direccionamiento indirecto:Las instrucciones acceden a una dirección de memoria que contiene la dirección de los datos reales.
Direccionamiento relativo:Las instrucciones acceden a una ubicación de memoria relativa a la ubicación actual de la instrucción.

9- ¿Qué función desempeña el zócalo de una placa base?

Su función es proporcionar una conexión física y eléctrica entre el procesador (CPU) y la placa base.
Permite que el procesador se instale y se retire fácilmente de la placa base sin dañarlo. Proporciona un mecanismo de sujeción seguro y asegura una conexión sólida y estable entre los pines del procesador y los contactos de la placa base.
