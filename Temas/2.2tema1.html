Tema 1: Unidades de medida

Ejercicios de unidades de medida

1- Desde qué año existen "nuevos prefijos informáticos"

Desde 1960


2- Ordenar por tamaño los siguientes archivos: 4000 KB, 0.3GB, 5MB, 0.02 GB, 400 KB, 0.5MB

400 KB, 0.5 MB, 5 MB, 4000 KB, 0.02 GB, 0.3 GB


3-¿En un USB de 16GB cuántos archivos de video de 500MB caben?

Primer: Tot en les mateixes unitats - Segon: Factor conversió: 16GB x 1 arxiu / 0.5GB
500MB\1024GB=0,5GB - 16GB\0,5GB = 32 videos
En un USB de 16GB caben aproximadamente 32 archivos de video de 500MB.


4- Cuántas fotos de 2 Megapixels = 2.000.000 bits caben en un CD (700 MB)

700MB * 1024 = 716,800KB - 716,800KB / 325KB = 2,203.07 foto
En un CD de 700MB caben aproximadamente 2,203 fotos de 325KB


5- Cuántas fotos de 2 Megapixels = 2.000.000 bits caben en un CD (700 MB)

Tamaño en bits = 2.000.000 píxeles * 8 bits/píxel = 16.000.000 bits
700MB * 8 bits/byte * 1024 kilobytes/MB * 1024 bytes/kilobyte = 5.824.000.000 bits
Número de fotos = Tamaño del CD / Tamaño de cada foto = 5.824.000.000 bits / 16.000.000 bits = 364 fotos (aproximadamente)
En un CD de 700MB caben aproximadamente 364 fotos de 2 Megapixels.


6- Un sistema informático de 80GB de disco duro, está ocupado en un 20%. ¿Cuántos CDs necesitaré para hacer una copia de seguridad? ¿Cuántos DVDs necesitaría?

-Espacio ocupado = 20% * 80GB = 0.2 * 80GB = 16GB
-Un CD normal tiene un almacenamiento de aproximado de 700MB o 0.7GB
Número de CDs = Espacio ocupado / Capacidad de un CD = 16GB / 0.7GB = 22.86 CDs

-Un DVD tiene una capacidad de almacenamiento de aproximado de 4.7GB.
Número de DVDs = Espacio ocupado / Capacidad de un DVD = 16GB / 4.7GB = 3.40 DVDs
Se necesitaria al menos 23 CDs o 4 DVDspara hacer una copia de seguridad.


7- Un archivo DIV-X ocupa 5,6GB. ¿Cuántos CD necesito para guardarlo? ¿Cuántos Blu-ray?

CD tipico 700MB de almacenamiento - Blu-ray tipico 25GB de almacenamiento
Número de CDs = Tamaño del archivo DIV-X / Capacidad de un CD = 5.6GB / 0.7GB =8 CD
Número de Blu-ray = Tamaño del archivo DIV-X / Capacidad de un Blu-ray (single-layer) = 5.6GB / 25GB = 0.224 Blu-ray
Se necesita al menos 8 CDs, 1 Blu-ray.


8- Sabiendo que un DVD de simple capa tiene una capacidad de 4,7 GB, que la capacidad de un CD es de 700 MB calcula a cuántos CDs equivale dicho DVD.

Capacidad de un DVD: 4.7 GB - Capacidad de un CD: 0.7 GB
Número de CDs = 4.7 GB / 0.7 GB - Número de CDs = 6.71 CDs
Un DVD equivale aproximadamente a 7 CDs


9- Suponiendo que 2 horas de cine en calidad Divx ocupan 700 MB, calcula cuántas películas podemos almacenar en un disco duro de 1,5 TB

1 TB = 1000 GB -1 GB = 1024 MB
Capacidad del disco duro: 1,5 TB = 1500 GB = 1500 * 1024 MB = 1536000 MB
Tamaño de una película en calidad Divx: 700 MB
Número de películas = 1536000 MB / 700 MB = 2194.29 películas
Se pueden almacenar aproximadamente 2194 películas en calidad Divx en un disco duro de 1,5 TB.


10- Suponiendo que una canción en formato mp3 ocupa 5 MB, calcula cuántas canciones podemos almacenar en un reproductor de mp3 con una memoria de 4 GB.

1 GB = 1024 MB - Capacidad del reproductor de mp3: 4 GB = 4 * 1024 MB = 4096 MB - Tamaño de una canción en formato mp3: 5 MB
Número de canciones = 4096 MB / 5 MB = 819.2 canciones
se pueden almacenar aproximadamente 819 canciones en formato mp3 en un reproductor de mp3 con una memoria de 4 GB.
